# Funnel Blueprint: VSL Funnel -- FitPro Accelerator

## Architecture Overview
- Funnel Type: 1 (VSL Funnel)
- Traffic Source: Paid (Meta/Instagram ads) + Organic (content retarget)
- Target ICP: Online fitness coaches making $3-8K/month
- Ticket Level: $1,497 (3x $547 payment plan)
- Estimated Conversion Path: 2.5% Ad CTR -> 35% Opt-In -> 55% VSL Watch (to CTA) -> 12% Application -> 65% Booked -> 80% Show -> 30% Close

## Funnel Map
```
[Meta Ad] -> Opt-In Page -> VSL Page -> Application -> Thank You
  CTR: 2.5%    Conv: 35%    Watch: 55%   Complete: 12%
                    |
              [Email Nurture: 5 emails]
                    |
              [Retarget Non-Completers]
```
