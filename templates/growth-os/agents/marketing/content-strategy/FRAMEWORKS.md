## Framework 2: Repurposing Brief

### Purpose
Map a single core content piece into 8-12 platform-native derivatives with specific adaptation instructions for each.

### Structural Blueprint

#### Section 1: Source Content Analysis
```
Source Platform: [YouTube | Podcast | Blog]
Source Title: [Title]
Source File/URL: [Path or link]
Core Message (1 sentence): [The single idea this piece communicates]
ICP Pain Point Addressed: [From icp.md]
Buyer Journey Stage: [Primary stage]
```

#### Section 2: Key Moment Extraction
```
| Timestamp / Section | Moment Type | Content | Derivative Potential |
|---------------------|-------------|---------|---------------------|
| [0:00-0:30] | Hook | "[Text]" | Reel hook, Tweet hook |
| [2:15-3:00] | Key Insight | "[Text]" | Standalone tweet, LinkedIn opening |
| [5:30-6:45] | Story Beat | "[Text]" | Story reel, Story sequence |
| [8:00-8:30] | Quotable | "[Text]" | Screenshot tweet, Story card |
| [12:00-13:30] | Framework | "[Steps]" | How-to thread, Framework LinkedIn |
```

#### Section 3: Derivative Production Briefs

For each derivative:
```
### [Platform] -- [Content Type]
Target Agent: [short-form | twitter | linkedin | stories]
Adaptation Angle: [How to reframe for this platform]
Hook: [Platform-native hook derived from source]
Key Points to Include: [Specific elements from source]
What to Change: [Tone shift, length compression, format change]
What to Keep: [Core message, specific phrases, data points]
CTA: [Platform-appropriate CTA]
Publish Date: [Staggered from source]
```

#### Section 4: Derivative Schedule
```
| Day | Derivative | Platform | Status |
|-----|-----------|----------|--------|
| Day 0 | Source publishes | [Platform] | -- |
| Day 0 | BTS story sequence | Instagram Stories | -- |
| Day 1 | Reel clip #1 (hook moment) | Instagram/TikTok | -- |
| Day 1 | Thread extraction | Twitter | -- |
| Day 2 | Reel clip #2 (key insight) | Instagram/TikTok | -- |
| Day 2 | LinkedIn reframe | LinkedIn | -- |
| Day 3 | Reel clip #3 (story beat) | Instagram/TikTok | -- |
| Day 3 | Standalone tweets (2-3) | Twitter | -- |
| Day 4 | Blog summary + SEO | Blog | -- |
| Day 5+ | Best-performer boosted as ad | Paid | -- |
```

### Quality Checklist
- [ ] Each derivative has a unique hook (not copied from source)
- [ ] Platform-native formatting applied (not copy-paste)
- [ ] Staggered publishing (no same-day source + all derivatives)
- [ ] Brand voice maintained across all adaptations
- [ ] Each derivative adds value beyond the source
- [ ] At least 8 derivatives produced from the source

## Framework 4: Campaign Content Plan

### Purpose
Design a multi-platform content sequence for a specific campaign or launch, with buildup, peak, and follow-through phases.

### Structural Blueprint

#### Section 1: Campaign Overview
```
Campaign Name: [Name]
Objective: [Awareness | Lead Gen | Sales | Launch]
Offer: [From offers.md]
Duration: [X days/weeks]
Buildup Phase: [Date range]
Peak Phase: [Date range]
Follow-Through Phase: [Date range]
```

#### Section 2: Phase-by-Phase Content Map

**Buildup Phase (5-7 days before peak):**
```
| Day | YouTube | Reels | Stories | Twitter | LinkedIn | Ads |
|-----|---------|-------|---------|---------|----------|-----|
| -7 | -- | Problem reel | Seed problem | Pain thread | -- | TOFU awareness |
| -6 | -- | -- | Personal story | Daily tweets | Story post | -- |
| -5 | Educational | Lifestyle | Hint solution | Hot take | -- | -- |
| -4 | -- | Testimonial | Social proof | Case study thread | Case study | MOFU retarget |
| -3 | -- | Hook+Value | Education | How-to thread | Framework | -- |
| -2 | -- | BTS | FAQ/Clarity | -- | Authority | -- |
| -1 | -- | Transformation | Anticipation | Tease | -- | BOFU priming |
```

**Peak Phase (launch/offer window):**
```
| Day | YouTube | Reels | Stories | Twitter | LinkedIn | Ads |
|-----|---------|-------|---------|---------|----------|-----|
| 0 | VSSL/Announce | Lead Magnet | Launch reveal | Announcement thread | Announcement | Full funnel live |
| +1 | -- | Testimonial | Objection break | Proof thread | Case study | Retarget engaged |
| +2 | -- | Quick Tips | FAQ | Daily + proof | -- | Retarget clickers |
| +3 | -- | -- | Skepticism removal | Hot take | -- | Urgency creative |
```

**Follow-Through Phase (after close):**
```
| Day | Content Focus |
|-----|---------------|
| +4 | Thank you + results recap across all platforms |
| +5 | Behind-the-scenes of the campaign |
| +6-7 | Return to value content, transition back to regular calendar |
```

#### Section 3: Cross-Platform Messaging Matrix
```
| Phase | Core Message | YouTube | Reels | Stories | Twitter | LinkedIn |
|-------|-------------|---------|-------|---------|---------|----------|
| Buildup | [Message] | [Adaptation] | [Adaptation] | [Adaptation] | [Adaptation] | [Adaptation] |
| Peak | [Message] | [Adaptation] | [Adaptation] | [Adaptation] | [Adaptation] | [Adaptation] |
| Follow | [Message] | [Adaptation] | [Adaptation] | [Adaptation] | [Adaptation] | [Adaptation] |
```

### Quality Checklist
- [ ] Campaign arc has clear buildup > peak > follow-through
- [ ] All active platforms participate in coordinated messaging
- [ ] Ad creative aligns with organic content themes
- [ ] Offer details from offers.md are accurately represented
- [ ] Urgency/scarcity elements are truthful and ethical
- [ ] Post-campaign transition back to regular calendar is planned
- [ ] Each platform plays its assigned role (not all doing the same thing)
